#### 说明
哈皮阅读是一款阅读器，主要用移动端的PDF阅读，  
使用场景：  
1.通过AirDrop(隔空投送)传给手机端的PDF文件，哈皮悦读进行接收并阅读；  
2.通过某应用(如微信)分享给哈皮悦读的PDF文件，跳转到哈皮悦读并阅读。
